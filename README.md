### [sinatra-builder](#)

A lightweight Sinatra skeleton builder

### Install

    curl -O https://raw.github.com/julianromera/sinatra-builder/master/build.sh  
    chmod u+x build.sh && ./build.sh __INSTALL_DIRECTORY__

### Structure

    myproject/
      public/
        public/css
        public/images
        public/javascript
      views/
      tmp/
      app.rb
      Gemfile
      config.ru
